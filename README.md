## 🚀 当前研究项目

### 🔐 基于 PyTorch 的验证码识别系统

> 一个从数据生成、标签编码到模型训练的完整深度学习工程实践项目。

### ✨ 项目特点

- 4位数字验证码自动生成  
- 干扰点与干扰线增强机制  
- 参数化配置管理（config.json）  
- 可扩展至字符型 / 文字型 / 图片型验证码  
- CNN → CRNN 结构升级规划

## 🏗 项目框架设计

### 📌 整体架构
```
验证码识别系统
│
├── 参数配置层
│ └── 统一实验参数管理（JSON）
│
├── 数据构建层
│ ├── 验证码图像生成
│ ├── 噪声增强策略
│ ├── 标签自动构建
│ └── 数据集划分
│
└── 实验准备层
├── 图像尺寸标准化预留
└── 模型训练参数预设
```
---

### 🎯 设计思想

- 模块化拆分，便于扩展升级  
- 数据生成与训练过程解耦  
- 支持从固定长度验证码向序列识别过渡  
- 为后续字符型 / 文字型 / 图片型验证码统一框架打基础  

---

### 🚀 可扩展方向

- 支持变长验证码识别  
- 引入 CTC Loss 处理序列问题  
- 增加更复杂干扰背景  
- 构建统一多类型验证码识别系统
