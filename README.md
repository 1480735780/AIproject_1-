# AIproject_1-

## 项目背景
随着互联网应用的快速发展，各类网站和系统普遍采用验证码机制，以防止恶意注册、暴力破解与自动化攻击。传统基于规则或模板匹配的识别方法，在复杂扭曲、噪声干扰和字符粘连等场景下识别效果有限，且泛化能力较弱。

近年来，深度学习在计算机视觉领域取得显著进展，尤其是卷积神经网络（CNN）和序列建模方法在图像识别任务中表现突出。因此，本项目计划基于 PyTorch 构建验证码识别模型，提升复杂图形验证码的识别准确率与鲁棒性，同时为 OCR 与智能识别相关技术的工程实践提供参考。

## 项目目标
- 构建可训练、可评估的验证码识别流程。
- 提升复杂干扰场景下的识别准确率。
- 沉淀可复用的数据处理与模型训练经验。
